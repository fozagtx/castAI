castAI provides Casper payment tooling for agent and HTTP workflows.

Packages:

| Package | Purpose |
|---|---|
| `@castaisdk/x402` | x402 exact payments on Casper |
| `@castaisdk/mpp` | MPP charge method for native CSPR transfers |
| `@castaisdk/facilitator` | x402 facilitator Worker |
| `@castaisdk/router` | x402 payment router Worker |

Supported networks:

| Network | Chain name | Asset |
|---|---|---|
| `casper:mainnet` | `casper` | `CSPR` |
| `casper:testnet` | `casper-test` | `CSPR` |
