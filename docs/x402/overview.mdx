---
title: x402 Overview
description: Casper exact payments for x402.
---

`@castaisdk/x402` adds Casper support to the x402 exact payment flow.

The client submits a native CSPR transfer and sends the deploy or transaction hash in the x402 payment payload. The facilitator verifies the hash through Casper RPC before settlement succeeds.

Verification checks:

| Check | Rule |
|---|---|
| Scheme | `exact` |
| Network | `casper:mainnet` or `casper:testnet` |
| Asset | `CSPR` |
| Amount | Required motes exactly |
| Recipient | Required public key or account hash |
| Execution | Casper RPC reports executed deploy data |
