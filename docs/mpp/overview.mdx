---
title: MPP Overview
description: Casper payment method for MPP.
---

`@castaisdk/mpp` exposes a `casper` charge method.

The method creates or accepts a native CSPR transfer hash, then verifies the transfer through Casper RPC.

Credential payloads:

```ts
{ type: "deployHash", deployHash: "..." }
{ type: "transactionHash", transactionHash: "..." }
```
