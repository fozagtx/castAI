# castAI Docs

Next.js App Router documentation site powered by Fumadocs.

## Develop

```sh
npm run dev
```

```sh
pnpm dev
```

```sh
yarn dev
```

```sh
bun run dev
```

The dev server prints its browser URL after startup.

## Content

Docs live in:

```txt
content/docs
```

Fumadocs generates the `.source` folder during `next dev` and `next build`.

## Build

```sh
npm run build
```

```sh
pnpm build
```

```sh
yarn build
```

```sh
bun run build
```

Static export output:

```txt
out
```
