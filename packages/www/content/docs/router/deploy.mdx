## Required environment

```txt
FACILITATOR_URL=https://your-facilitator.example
CASPER_MAINNET_PAY_TO=
CASPER_TESTNET_PAY_TO=
CASPER_MAINNET_PRIVATE_KEY=
CASPER_TESTNET_PRIVATE_KEY=
```

## Runtime

Run the router package in the HTTP runtime you choose. The service needs the
environment values above and outbound access to the facilitator and destination
URLs it forwards.

## Runtime checks

Before using the router with money:

1. Use a funded Casper testnet private key.
2. Use a real x402 destination that returns `402 Payment Required`.
3. Confirm the router returns Casper payment requirements.
4. Pay the router request.
5. Confirm the destination receives the paid request.
6. Confirm the Casper deploy or transaction exists on the configured RPC.
