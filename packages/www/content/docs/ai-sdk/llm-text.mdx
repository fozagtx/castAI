---
title: LLM Text
description: Convert paid resource responses into plain text for model context.
---

`llm.text` formats a paid resource response into a stable plain-text block for an LLM.

Use it when your app fetches a paid resource outside an AI SDK tool call, then needs to pass the result into a prompt or message.

```ts
import { fetchResource, llm } from "@castaisdk/ai-sdk";

const response = await fetchResource(x402Fetch, {
  url: "https://api.example.com/protected",
});

const text = llm.text(response, {
  includeHeaders: false,
  maxBodyCharacters: 12_000,
});
```

Output shape:

```txt
URL: https://api.example.com/protected
Status: 200 OK
OK: true
Content-Type: application/json

Body:
{ ... }
```

For one-shot usage:

```ts
import { getPaidResourceText } from "@castaisdk/ai-sdk";

const text = await getPaidResourceText(x402Fetch, {
  url: "https://api.example.com/protected",
});
```

This helper does not create or invent payment data. It only formats the actual paid HTTP response.

## Docs AI Chat

The docs site also exposes a Fumadocs Ask AI panel. It uses the Markdown docs routes and the server-side docs search tool before answering.

Configure the server route with:

```txt
OPENROUTER_API_KEY=
CASTAI_DOCS_SITE_URL=
CASTAI_DOCS_AI_MODELS=
```

`CASTAI_DOCS_AI_MODELS` is optional. When it is empty, the server uses the built-in free fallback pool. The public UI does not display model names.
