## One-call text agent

```ts
import { generateCastaiText } from "@castai/ai-sdk";

const result = await generateCastaiText({
  model: "openai/gpt-4.1",
  x402: {
    networks: ["casper:testnet"],
    privateKeyPem: process.env.CASPER_PRIVATE_KEY_PEM,
  },
  prompt: "Fetch the paid weather endpoint and summarize the JSON.",
});

console.log(result.text);
```

`generateCastaiText` uses AI SDK `generateText`, provides the castAI tools, and defaults to `stepCountIs(5)` so the model can call a payment tool and then write the final answer.

## x402 agent with explicit tools

```ts
import { generateText } from "ai";
import {
  createCastaiAgentTools,
  createCasperX402Fetch,
} from "@castai/ai-sdk";

const x402Fetch = createCasperX402Fetch({
  networks: ["casper:testnet"],
  privateKeyPem: process.env.CASPER_PRIVATE_KEY_PEM,
});

const result = await generateText({
  model: "openai/gpt-4.1",
  tools: createCastaiAgentTools({
    x402: { fetch: x402Fetch },
  }),
  prompt: "Fetch the paid weather endpoint and summarize the result.",
});

console.log(result.text);
```

## MPP agent

```ts
import { generateText } from "ai";
import {
  createCastaiAgentTools,
  createCasperMppFetch,
} from "@castai/ai-sdk";

const mppFetch = createCasperMppFetch({
  network: "casper:testnet",
  privateKeyPem: process.env.CASPER_PRIVATE_KEY_PEM,
});

const result = await generateText({
  model: "openai/gpt-4.1",
  tools: createCastaiAgentTools({
    mpp: { fetch: mppFetch },
  }),
  prompt: "Fetch the MPP protected endpoint and return the JSON payload.",
});

console.log(result.toolResults);
```

## Combined agent

```ts
const tools = createCastaiAgentTools({
  x402: { fetch: x402Fetch },
  mpp: { fetch: mppFetch },
});
```

The model can choose `payX402Resource` or `payMppResource` based on the URL or instruction.

## Agent factory

```ts
import { createCastaiAgent } from "@castai/ai-sdk";

const agent = createCastaiAgent({
  x402: {
    networks: ["casper:testnet"],
    privateKeyPem: process.env.CASPER_PRIVATE_KEY_PEM,
  },
});

// agent.tools works with generateText.
// agent.system can be appended to your existing system prompt.
```

## Required environment

```txt
CASPER_PRIVATE_KEY_PEM=-----BEGIN PRIVATE KEY-----
```

Use funded testnet keys for testnet payment tests.
