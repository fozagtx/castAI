---
title: Facilitator
description: Verify and settle Casper x402 payment hashes.
---

The facilitator verifies Casper payment hashes through RPC and returns x402 settlement responses.

Settlement responses are built from the submitted payment hash and the configured Casper RPC.

## Required environment

```txt
CASPER_MAINNET_RPC_URL=https://node.mainnet.casper.network/rpc
CASPER_TESTNET_RPC_URL=https://node.testnet.casper.network/rpc
CASPER_FACILITATOR_SIGNERS=
```

`CASPER_FACILITATOR_SIGNERS` is optional. Use it only when the service must restrict accepted sender public keys.

## Verification checks

- Hash exists on the configured Casper RPC.
- Transfer is native CSPR.
- Execution is confirmed and not failed.
- Recipient matches `payTo`.
- Amount matches the x402 payment requirement in motes.
- Sender matches configured signer rules when signer rules exist.

## Runtime

Run the facilitator package in the HTTP runtime you choose. The service needs
the environment values above and outbound access to the configured Casper RPC.
