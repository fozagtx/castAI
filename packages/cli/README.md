# @castai/cli

CLI for castAI scaffolding and MCP configuration.

## Install

```sh
npm install -g @castai/cli
```

## Commands

```sh
castai templates
castai scaffold next-checkout ./my-checkout --package-manager npm
castai scaffold agent-vercel-ai ./my-agent --package-manager pnpm
castai scaffold mcp-claude-code ./claude-code-castai --package-manager bun
castai claude-code --json
castai doctor --json
```
